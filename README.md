# 💰 SmartTracker AI — Smart Expense Tracker

An AI-powered personal finance tracker built with pure HTML, CSS & JavaScript.
No installations needed. Just open `index.html` in any browser!

---

## 🚀 How to Open

1. Double-click `index.html`
2. Opens in your browser instantly
3. Login with any email & password (demo mode)

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔐 Login Page | Beautiful sign-in with email/password |
| 🏠 Dashboard | Balance, income, expenses, charts |
| 💳 Transactions | Search, filter, delete, export |
| 📊 Budget Planner | Set & edit monthly budgets |
| 🧾 Bill Reminders | Mark bills paid/unpaid |
| 📈 Analytics | Area chart, pie chart, statistics |
| 🏦 Accounts | Multi-account net worth view |
| 🎯 Goals | Create savings goals |
| 🤖 AI Insights | Smart spending alerts |
| 💬 AI Chat | Ask questions about your finances |
| ⚙️ Settings | Dark mode, notifications, export |
| 📤 CSV Export | Downloads real .csv file |
